# Ragmir UI

Unpublished workspace package for shared Ragmir React primitives and Tailwind theme tokens.

It is adapted from the WorkoutGen UI/landing approach, but it contains Ragmir-specific styling only:
no WorkoutGen copy, assets, analytics, CDN configuration, or secrets.

Used by:

- `@jcode.labs/ragmir-landing`

```bash
pnpm --filter @jcode.labs/ragmir-ui check
```
